# Телеграм-канал: цепочка работы

Проект нужен, чтобы готовить тексты и план публикаций для телеграм-канала.

Сейчас главный упор — на идеи, посты и расписание. Автоматическая отправка в канал будет добавлена позже.

```mermaid
flowchart TD
    A["Идея канала"] --> B["Исследование темы"]
    B --> C["Файл Исследование.md"]
    C --> D["План публикаций.md"]
    D --> E["Идея поста на день"]
    E --> F{"Смысла хватает?"}
    F -->|"Да"| G["Агент пишет черновик"]
    F -->|"Нет"| H["Агент задает до 3 коротких вопросов"]
    H --> G
    G --> I["Пост сохраняется в папку Посты"]
    I --> J["Статус: черновик"]
    J --> K{"Текст одобрен?"}
    K -->|"Нет"| L["Правки"]
    L --> G
    K -->|"Да"| M["Статус: готово"]
    M --> N["Пост ставится в расписание"]
    N --> O["Публикация в канал"]
    O --> P["Статус: опубликовано"]
```

Пока агент не публикует посты сам. Он готовит тексты, сохраняет их в файлы и помогает вести план.
